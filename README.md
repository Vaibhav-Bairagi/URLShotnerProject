# URL Shortener API

This project is a URL Shortener built using ASP.NET Core Web API and SQL Server.

It allows users to convert long URLs into short, unique links and redirect back to the original URL when accessed.

## Features
- Generate short URLs from long URLs
- Redirect short URL to original URL
- Store URL mappings in SQL Server database
- Clean and scalable Web API architecture

## Technologies Used
- ASP.NET Core Web API
- C#
- SQL Server
- Entity Framework Core

## How to Run
1. Clone the repository
2. Update database connection string in appsettings.json
3. Run database migrations
4. Start the API using Visual Studio or dotnet CLI
